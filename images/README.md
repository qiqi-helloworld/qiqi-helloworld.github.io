## 图片要求:
- 推荐尺寸: 160x160像素或更大 (保持方形比例)
- 格式: PNG (带透明背景更佳)
- 如果某篇论文没有图片,网页会自动显示您的头像作为占位符

## 如何添加图片:
1. 从论文PDF、GitHub仓库或项目页面中截取/下载图片
2. 使用图片编辑工具裁剪为正方形
3. 调整大小为160x160或320x320像素
4. 保存为对应的文件名到这个文件夹

## 提示:
- ABSGD、SOAP、RECOVER等GitHub仓库的README通常都有示意图
- 可以使用论文中的Figure 1(通常是模型概览图)
- 截图后可以用Preview或其他工具裁剪和调整大小
